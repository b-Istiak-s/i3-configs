# i3 configuration

Personal i3 window manager configuration for Arch Linux.

This repository contains my i3 setup, related configs, and small scripts used in my daily workflow.

## 1. About the configs

These configs are my personal setup for running i3 on Arch Linux.  
The setup is focused on keyboard usage, simplicity, and low resource usage.

## 2. What I have added

1. Custom keybindings  
2. Polybar configuration  
3. Helper shell scripts  
4. Brightness, volume, Bluetooth modules  
5. Workspace and layout rules  

## 3. What I plan to add next

## 5. Inspiration to move to i3

I moved to i3 to improve productivity and reduce distractions.  
The goal was to use fewer graphical tools and rely more on the keyboard. 
I had already used Hyprland for a couple of months, and I really enjoyed it. However, I wanted something lightweight. I had xfce4 too at the time, which I used whenever I needed to save battery. 
Still, when I was using xfce4, I noticed my laptop's fan would start running rapidly most of the time, so I decided to start afresh with i3 (I randomly installed SO MANY packages, so I was running low on storage (HDD got destroyed too)).
